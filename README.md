## Userbot + Assistant for Telegram

Currently work fine perfectly, but some features may give you error. Please report a bug to me if you facing any issues.

## Installation Guide

[Visit the official documentation](https://aman-a.gitbook.io/nana-remix) to learn more how to setup your own Nana userbot + assistant.

## Commands List

TO get all the available commands for the userbot + assistant, send `/help` on your Assistant.

## Community

Have questions about our fork of Nana or facing issues? Join [Nana Support](https://t.me/NanaBotSupport) and our friendly community will answer you some questions.

For support to the forks you're using, please refer to their documentation or if they're in the support chat,
please mention them about your issue.
